Problem 1: Find the maximum consecutive 1s in a binaryArray

Solution: Algorithm Used

    I initialized maxCount = 0 to store the maximum number of consecutive 1s counted.
    I initialized currentCount = 0 to count numbers of consecutive 1s at a given instance.
    I looped through the array: if the current element is 1, currentCount is increamented. If the current count is 0, maxCount is updated with the highest count between maxCount and currentCount before currentCount is reset to 0.
    I updated maxCount one more time after the loop to capture maximum consecutive 1s if the longest sequence ends the array.
    I returned maxCount to output the maximum consecutive 1s in the array.
